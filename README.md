# 🚀 AI Career Copilot: Resume Analyzer

AI Career Copilot is a powerful web application designed to help job seekers optimize their resumes using AI. It evaluates your resume against specific career goals, identifies skill gaps, and generates a personalized learning roadmap.

## ✨ Features

- **AI-Powered Analysis**: Uses Mistral AI to evaluate resumes based on user-defined goals.
- **Skill Extraction**: Automatically extracts relevant skills from uploaded PDFs or Docx files.
- **Gap Identification**: Identifies missing skills required for your target role.
- **Personalized Roadmap**: Generates a step-by-step learning path to bridge skill gaps.
- **Interview Prep**: Provides a curated list of interview questions tailored to your profile.
- **History Tracking**: Securely saves all your previous analyses for future reference.

## 🛠️ Technology Stack

- **Backend**: Python, Flask, SQLAlchemy
- **Database**: SQLite
- **AI Integration**: Mistral AI API
- **Frontend**: HTML5, Modern CSS (Glassmorphism & Dynamic Gradients)
- **Document Processing**: PyPDF2, python-docx

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- A Mistral AI API Key

### AI Resume Checker 
 Available at your primary URL https:
 ```bash
    https://ai-project-resume-checker-1.onrender.com
   ```


### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/laganmehra875/AI-PROJECT-RESUME-CHECKER.git
   cd AI-PROJECT-RESUME-CHECKER
   ```

2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Set up your environment variables:
   Create a `.env` file in the root directory and add your Mistral API Key:
   ```env
   MISTRAL_API_KEY=your_api_key_here
   ```

5. Run the application:
   ```bash
   python app.py
   ```

6. Open your browser and navigate to `http://127.0.0.1:5000`.
